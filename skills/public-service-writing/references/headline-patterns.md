# 标题技法

## 八种技法

| 技法 | 公式 | 示例 | 适用原型 |
| --- | --- | --- | --- |
| 口语祈使 | 求你了 + 别做X | "求你了，别再用开水烫碗了" | 劝诫 |
| 震惊悬念 | 结果/数字 + 竟然是因为X | "姑娘住进ICU，竟然是因为一杯奶茶" | 劝诫、辟谣 |
| 数字冲击 | 具体数字 + 后果 | "一年吃掉12斤农药？真相是……" | 辟谣、科普拟人 |
| 第二人称质问 | 你是不是也在X？ | "你是不是也在用错方式洗内裤？" | 劝诫、时令 |
| 拟人独白 | 器官/细胞 + 第一人称 | "我是你的肝，今天想跟你谈谈" | 科普拟人 |
| 时令锚定 | 节气/节日/天气 + 提醒 | "三伏天到了，这件事比中暑更危险" | 时令 |
| 反转否定 | X不是Y，而是Z | "你以为的养生，其实在养病" | 辟谣、劝诫 |
| 紧急通知 | 紧急/注意 + 事件 + 行动 | "紧急提醒：这批次退烧药已召回" | 应急 |

## 标题红线

1. **不伪造医学紧急**——不把非紧急话题包装成"紧急""速看""救命"
2. **不羞辱特定群体**——标题不以性别、年龄、体型、疾病做贬义标签
3. **不用无法核实的死伤数字**——未经官方确认的伤亡数据不入标题
